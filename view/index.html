Route HTTP Method Description
/ GET Serves the index.html file
/login POST Logs in a user
/users GET Displays all users
/user/:id GET Displays a single user by ID
/user/:id PUT Updates a user by ID
/register POST Creates a new user
/user/:id DELETE Deletes a user by ID
/products GET Displays all products
/product/:id GET Displays a single product by ID
/product/:id PUT Updates a product by ID
/product POST Creates a new product
/product/:id DELETE Deletes a product by ID
/orders GET Displays all orders
/order/:id GET Displays a single order by ID
/order/:id PUT Updates a order by ID
/order POST Creates a new order
/order/:id DELETE Deletes a order by ID
